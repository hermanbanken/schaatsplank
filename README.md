# Schaatsplank

App op je Android telefoon die:
- meet of en hoe snel je schaatsslagen doet
- meet hoe diep je zit
- een HTTP server opzet
- een WebSocket server opzet
- de gegevens deelt via de websocket
- de resultaten onthoudt
